# AI拼图字段捷径

这是一个多维表格字段捷径，可以根据用户选择的图片附件自动生成AI拼图。

## 功能特性

- **图片附件选择**：支持选择多个图片附件作为拼图素材（必填）
- **尺寸选择**：支持选择拼图尺寸比例（3:4、2:3、9:16），默认为3:4（非必填）
- **标题输入**：支持为生成的拼图添加标题（非必填）
- **自动下载**：生成的拼图会自动下载为附件显示在表格中
- **数量限制**：如果生成的拼图超过5张，只保存前5张

## 使用方法

1. 在多维表格中创建字段时选择"字段捷径"
2. 搜索并选择"AI拼图"捷径
3. 配置参数：
   - 选择图片附件（必填）
   - 选择图片尺寸（可选，默认3:4）
   - 输入图片标题（可选）
4. 保存字段，系统会自动调用AI拼图接口生成拼图

## 开发说明

### 项目结构
```
├── config.json          # 本地调试配置文件
├── package.json         # 项目依赖配置
├── tsconfig.json        # TypeScript配置
├── src/
│   └── index.ts         # 主要代码文件
└── README.md           # 说明文档
```

### 本地调试

1. 安装依赖：
```bash
npm install
```

2. 启动本地服务：
```bash
npm run start
```

3. 使用字段捷径调试助手进行调试

### 发布

1. 打包项目：
```bash
npm run pack
```

2. 上传 `output/output.zip` 文件到多维表格平台

## 技术实现

- 使用 `@lark-opdev/block-basekit-server-api` 框架
- 调用AI拼图接口
- 支持多种图片尺寸比例
- 自动处理图片下载和附件生成
- 完善的错误处理和日志记录

## 注意事项

- 确保选择的图片附件格式正确
- 生成的拼图数量可能因模板而异
- 如果接口返回超过5张图片，只保存前5张
- 建议在稳定的网络环境下使用
